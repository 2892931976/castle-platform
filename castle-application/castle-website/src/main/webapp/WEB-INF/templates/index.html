关于我们

广州当凌信息科技有限公司成立于2015年8月，是一家提供软件服务和软件产品的公司。公司目前拥有专业成熟的运营团队以及具备独立研发能力和快速响应能力的技术服务团队。

移动城堡是当凌信息科技有限公司的主要产品，致力于帮助人们实现移动信息化。首创以互联网的方式来管理设备和应用。我们坚持以用户体验为最高诉求，承诺对付费会员7*24小时服务，竭力为每个会员带来独一无二的技术服务体验。
